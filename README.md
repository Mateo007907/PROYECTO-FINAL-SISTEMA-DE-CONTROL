# PROYECTO-FINAL-SISTEMA-DE-CONTROL
El proyecto consiste en un sistema automático de detección de fugas de gas que utiliza un sensor MQ-4 y un microcontrolador PIC16F887. Cuando se detecta una concentración peligrosa de gas, el sistema activa un relé que enciende un motor extractor, un buzzer y un LED, alertando y ventilando el ambiente para prevenir accidentes.
